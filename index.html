O que é o GIT?

É uma ferramenta que salva o histórico das mundançãs em arquivos. ele permite testar versões e alterações.

para que server o GITHUB?

É uma plataforma usando para gudar, orgaizar e compartilhar código na internet, usando o GIT.

para que server o GIT clone?

ele faz uma cópia completa de um repositório do GITHUB para você trabalhar localmente

para que server o GIT add?

Server para selecionar e preparar as alterações feitas nos arquivos que eles possam ser salvar no histórico do GIT de um commit

para que server o GIT commit

server para salvar de vez as alterações que você preparou com git add no histórico do projeto, criando uma versão registrada com uma mensagem explicando o que mudou

para que server o GIT commit

server para enviar os commits que você salvou no seu computador para um repositório remoto, como no GIhub, deixando o projeto oline e atualizado para todos.
